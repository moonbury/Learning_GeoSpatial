Urban area-sorted files are available as fixed format text files and Excel files. Each urban area-sorted list file is sorted by urban area census code (UACE) and contains records for either: 1) all urban areas (ua_list_all.txt/xls), 2) urbanized areas (ua_list_ua.txt/xls), or 3) urban clusters (ua_list_uc.txt/xls). Each file includes urban areas in the United States, Puerto Rico, and the Island Areas.

The text files are fixed format and designed for internet viewing. Five blanks are included after each field in the fixed format file. Start and end positions are included in the layout below for the fixed format text files. Area measurements are presented in both square meters and square miles. Area measurements in square miles are rounded to two decimal places. Population density values are in persons per square mile of land area, rounded to one decimal place.
